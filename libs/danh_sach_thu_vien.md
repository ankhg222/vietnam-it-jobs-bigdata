# Danh sách các phần mềm và thư viện liên quan

## 1. Phần mềm và Công nghệ lõi
- **Apache Hadoop (HDFS, MapReduce)**: Hệ thống lưu trữ phân tán và xử lý dữ liệu lớn.
- **Apache Spark**: Framework xử lý dữ liệu lớn trên bộ nhớ.
- **Python 3.x**: Ngôn ngữ lập trình chính cho Crawler, Machine Learning và Web API.
- **Hive**: Data warehouse phục vụ truy vấn dữ liệu (nếu có dùng chung với Hadoop).

## 2. Thư viện Python (Backend & Phân tích dữ liệu)
- **FastAPI / Uvicorn**: Xây dựng và triển khai Web API hiệu năng cao cho hệ thống gợi ý.
- **Pydantic**: Ràng buộc kiểu dữ liệu cho API.
- **Pandas / Numpy**: Tiền xử lý, thao tác và tính toán trên dữ liệu dạng bảng, ma trận.

## 3. Thư viện Machine Learning & NLP
- **Scikit-learn**: Thuật toán máy học truyền thống, tính toán độ tương đồng (Cosine Similarity).
- **XGBoost**: Mô hình Gradient Boosting cho các tác vụ dự đoán nâng cao.
- **Sentence-Transformers**: Chuyển đổi văn bản đa ngôn ngữ thành vector nhúng (Embeddings) để phục vụ cho Content-Based Recommender System.
- **Mlxtend**: Khai phá luật kết hợp (Association Rule Learning - thường dùng với Market Basket Analysis/Skill Co-occurrence).

## 4. Thư viện Trực quan hóa
- **Matplotlib / Seaborn**: Vẽ biểu đồ, thống kê dữ liệu.
- **NetworkX**: Phân tích và biểu diễn mạng lưới (ví dụ: mạng lưới kỹ năng liên quan).
- **WordCloud**: Trực quan hóa tần suất xuất hiện của từ khóa/kỹ năng.
